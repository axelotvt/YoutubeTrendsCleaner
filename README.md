# YouTube Trends CSV Cleaner

A fast and simple tool to clean the CSV files downloaded from **Google Trends (YouTube)** so you can easily **copy and paste keywords** into your YouTube videos.

This tool removes Google Trends metadata, dates, categories, percentages, and other unwanted values, leaving you with **clean search queries only**.
